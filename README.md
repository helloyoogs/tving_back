サブスクリプションサービスプラットフォーム

2023.04.01 ~ 2023.05.01

Netflix・TVINGのようなサブスクリプション型動画サービスサイト

技術スタック :

- FRONT : React, TypeScript

- BACK : Spring Boot, Java, JPA, Spring Security, JWT, RESTful API

- ETC : Redis

Github (FRONT) : https://github.com/helloyoogs/tving_front

Github (BACK) : https://github.com/helloyoogs/tving_back

デモ動画 : https://youtu.be/pCAG_vlBPzI

実装機能 :

- ログイン、ログアウト、会員登録

- サブスク商品決済、アップグレード、ダウングレード

- 定期購読のキャンセル・再開

- 現在および翌月の購読商品確認

- 利用履歴照会

- お気に入り機能・お気に入りリスト照会
